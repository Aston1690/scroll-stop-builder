# Scroll-Stop Builder

A Claude Code skill that takes a video file and builds a production-quality website with Apple-style scroll-driven animation. The video plays forward/backward as the user scrolls.

## One-Line Install

```bash
curl -sL https://raw.githubusercontent.com/Aston1690/scroll-stop-builder/main/install.sh | bash
```

## Usage

Open Claude Code and type:

```
/scroll-stop-builder
```

Then provide a video file and answer a few questions about your brand.

## Requirements

- [Claude Code](https://claude.com/claude-code) installed
- FFmpeg (`brew install ffmpeg`)
- A short video (3-10 seconds), starting on a white background
